# Csv_To_Excel
这是一个便捷的将CSV转换为EXCEL并且可以复制的工具！！！
你无需新建一个csv文件，以txt文件打开编辑再把内容复制到excel中！
一个无需安装依赖的本地应用方案，包含浏览器版和 Windows 双击启动版。

## 功能

- 将 CSV 文本粘贴到输入框后即时转换为表格预览
- 支持带引号、逗号、换行的标准 CSV 内容
- 右侧生成可直接选中复制的表格
- 提供“复制整表”按钮，便于粘贴到 Word、Excel、WPS

## 使用方式

浏览器版：

直接在浏览器打开 [index.html](./index.html) 即可。

Windows 双击启动版：

进入 [desktop-app](./desktop-app) 目录，双击 `启动应用.bat` 即可。

## 操作说明

1. 在左侧输入框粘贴 CSV 文本
2. 点击“转换”
3. 在右侧直接框选需要的单元格复制，或点击“复制整表”
4. 粘贴到 Word、Excel 或 WPS 中使用
