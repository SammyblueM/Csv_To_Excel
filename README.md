# Chrome 插件版

这是一个 Chrome 浏览器扩展。

## 安装方式

1. 打开 Chrome，进入 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录：`F:\CsvToExcel\chrome-extension`

## 使用方式

1. 点击浏览器工具栏中的扩展图标
2. 打开 `CSV 转 Excel 预览`
3. 粘贴 CSV 文本
4. 点击“转换”
5. 点击扩展图标后，选择“右侧边栏”或“页面底部”
6. 工作区会直接嵌入当前网页
7. 直接复制右侧表格，或点击“复制整表”

## 说明

- 不需要 nginx
- 不需要后端服务
- 扩展会把工作区嵌入到当前网页中
- 支持右侧边栏和底部面板两种展示方式
- `chrome://`、扩展页、Chrome 应用商店这类特殊页面通常不能注入，这是 Chrome 本身限制
